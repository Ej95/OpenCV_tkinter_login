# OpenCV 結合學生資料二步驟解鎖

### 目的 : 藉由 Tkinter 結合 OpenCV & MYSQL 達成使用者登入畫面

### 使用方法 :memo: 

#### 1. git clone 到電腦內

#### 2. 將 mysql 串接到自己設置的資料庫

#### 3. 將圖片丟到 my_head 資料夾中，按照數字順序命名 （要配合迴圈使用）

#### 4. 丟完之後，切換到 train.py 執行並開始訓練模型

#### 5. 回到 tkinter-1.py 執行就可以了~
